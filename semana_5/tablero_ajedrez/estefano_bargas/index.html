Tarea Grupal

Para este ejercicio contamos con un div que contiene la clase board
La tarea es poder generar un tablero de ajedrez dentro de ese div, utilizando JavaScript.

Debejos crear cada cuadrado de forma tal que reprecente el espacio de una casilla blanca o negra según corresponda.
Entonces mediante la creación dinámica de div's vamos a rellenar el espacio (serían un total de 64 divs donde 32 tendrán el color negro y los otros 32 de color blanco).
Podemos utilizar las clases black y white las que ya contienen el color de fondo necesario.

Si necesitas observar como se tendría que ver el trabajo final, podes hacer click en el botón.
